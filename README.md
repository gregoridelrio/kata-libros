## Vas a gestionar una biblioteca con varios libros.
Cada libro debe tener:

Título
Autor
Año de publicación
Género (Novela, Ensayo, Fantasía, Ciencia Ficción, etc.)
Número de páginas
Número de veces prestado

Registra al menos 5 libros en un array.

Luego implementa dos funciones:

1️⃣ Función que devuelva el libro más prestado
Debe recorrer la lista y devolver el libro con mayor número de préstamos.

2️⃣ Función que devuelva los libros “clásicos”
Consideraremos clásico cualquier libro publicado antes del año 2000.
La función debe devolver un array con todos los libros que cumplan esa condición.
